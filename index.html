hgfreenet

 

What Is It?

This is an extension for Mercurial that provides utilities for serving repositories and bundles over Freenet (an anonymous p2p network).

Basic Features

  • Any repository uploaded to a Freenet URI with this extension can be accessed via 'hg clone|pull static-http://127.0.0.1:8888/freeneturi', so this extension is *not* required to read the repositories in Freenet.
  • A FCP protocol handler for 'hg clone|pull|push fcp://127.0.0.1:9481/freeneturi'
  • additional commands:
    • hg fcp-bundle -> wraps around hg bundle and put the bundle to a Freenet URI instead a local file
    • hg fcp-unbundle -> wraps around hg unbundle and get the bundle from a Freenet URI instead a local file
    • hg fcp-updatestatic -> upload the repository to Freenet, using some fallback code to work also at worst case (an ill or naked Freenet node e.g.)
